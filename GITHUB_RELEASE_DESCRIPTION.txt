## 🎉 Session Tracker v1.2.0

Professional desktop session tracking with advanced countdown timer and intelligent alert system.

---

### ⭐ What's New

#### 🕐 Advanced Countdown Timer
- **Millisecond Precision**: Real-time countdown with `HH:MM:SS.mmm` format
- **Customizable Duration**: Set timers up to 23 hours, 59 minutes, 59 seconds
- **Visual Display**: Clean, centered countdown with professional styling
- **Smooth Updates**: 50ms refresh rate for seamless countdown experience

#### 🔔 Intelligent Alert System
- **Sound Notifications**: WAV audio playback with 3x repeat on timer expiration
- **Visual Alerts**: Clear pop-up dialogs when countdown completes
- **System Tray Notifications**: Unobtrusive desktop notifications
- **Auto-Stop Sessions**: Automatically ends session when timer reaches zero

#### ⚡ Enhanced Session Management
- Timer properly integrates with pause/resume functionality
- Microsecond-level accuracy for precise time tracking
- Timer configuration locks during active sessions
- Smart validation prevents invalid timer settings

---

### 📦 Installation

#### Quick Start
```bash
# Extract the archive
unzip session-tracker-v1.2.0.zip
cd release-v1.2.0

# Install dependencies
pip install -r requirements.txt

# Run application
python session_tracker.py
```

#### Requirements
- **Python**: 3.8 or higher
- **PyQt6**: GUI framework (required)
- **PyQt6-Multimedia**: Sound support (required)
- **reportlab**: PDF generation (required)
- **PyQt6-Charts**: Charts functionality (optional)

---

### 🎯 Core Features

- ✅ **Session Tracking**: Start, pause, resume, and stop work sessions
- ✅ **Timer System**: Countdown timer with millisecond precision and alerts
- ✅ **Calendar View**: Color-coded calendar showing session activity
- ✅ **Statistics**: Comprehensive analytics and session insights
- ✅ **Charts**: Interactive pie and bar charts (requires PyQt6-Charts)
- ✅ **PDF Reports**: Generate weekly and monthly session reports
- ✅ **CSV Export/Import**: Easy data portability with backup system
- ✅ **System Tray**: Minimize to tray with quick access controls
- ✅ **Keyboard Shortcuts**:
  - `Ctrl+Shift+S` - Start Session
  - `Ctrl+Shift+T` - Stop Session
  - `Ctrl+Shift+P` - Pause/Resume
  - `Ctrl+Shift+E` - Export Data

### ⚠️ Known Limitations

- Sound alerts require WAV format sound files (falls back to system beep)
- Charts functionality requires optional PyQt6-Charts package
- PDF reports use letter page size (US standard)
- Maximum timer duration: 23h 59m 59s
